L'application compilée depuis le code initial est dans le fichier "ApplicationGestionReservation.jar".

La commande pour l'executer en ligne de commande est : "java -jar ApplicationGestionReservation.jar"

Le système est une liste de choix numérotée, il faut entrer le numéro correspondant à la tâche à effectuer.

A l'exécution, il faut choisir entre le volet Client et le volet Administrateur (1 ou 2).
